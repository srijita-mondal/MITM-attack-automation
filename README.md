# MITM Attack Automation Framework

This project implements basic Man-in-the-Middle (MITM) attack workflows including ARP spoofing and DNS spoofing to intercept and observe network traffic in a controlled laboratory environment. The goal is to understand network protocol behavior and common trust assumptions in local network communication.

The framework provides simple automation scripts for setting up MITM positioning between hosts and capturing redirected traffic for analysis. The implementation is intended for educational and cybersecurity research purposes.

## Features
- ARP spoofing for traffic redirection
- DNS spoofing demonstration
- Packet interception and inspection
- Linux-based testing environment
- Scripted attack setup
