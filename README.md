# MITM Attack Automation Framework

This project demonstrates automated Man-in-the-Middle (MitM) attack techniques in a controlled virtual lab using Kali Linux and Metasploitable.

Implemented components include:

- ARP spoofing using Scapy and arpspoof
- DNS spoofing using Bettercap
- SSL stripping via Bettercap HTTP proxy
- Packet sniffing using Wireshark
- Python-based ARP spoof automation

All experiments were performed in an isolated virtual environment for educational and security research purposes.
